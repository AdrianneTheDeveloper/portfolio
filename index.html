Decoder Ring

Decoder Ring is a web application that uses 3 distinct cyphers (Caesar, Polybius, and Substitution). Based on the user's selection and input, each cypher will either encode or decode their inputted message.

Skills used: Javascript

I wrote the Javascript responsible for functionality of the caesar, substitution, and polybius cyphers for this site.

Project: Local Library

Project: Local Library is a library application where people can lend and borrow books.

Skills used: Javascript

I built the functionality of the dashboard to a local library application that shows which books are available, which are currently out, and other general statistics about the program.
